The Skeleton boilerplate

Skeleton is a super simple css boilerplate that consist of a single stylesheet and a set of rules. It gives you a responsive web-page with beautiful minimalistic styles that applies to a basic grid, buttons, forms and table elements. From this template you can go on and develop your own design, but it's a start without the pain of having to write all the CSS code for a basic responsive web-page.
